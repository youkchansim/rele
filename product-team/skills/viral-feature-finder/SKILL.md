# Viral Feature Finder

Marketer exclusive skill. Design natural sharing and viral loops within the product.

## Viral Loop Framework

```markdown
## Viral Analysis: [Feature Name]

### 1. Viral Type

- [ ] **Natural Viral**: Naturally exposed during usage (e.g., shared schedule)
- [ ] **Incentivized Viral**: Reward for sharing (e.g., invite friend -> premium feature)
- [ ] **Content Viral**: Created content gets shared (e.g., weekly report image)
- [ ] **Social Proof**: Usage itself is promotion (e.g., "Powered by [App]")

### 2. K-factor Analysis

K = i x c
- **i (invitations)**: Average invites/shares per user
- **c (conversion)**: Install/signup rate of invitees
```

## Build in Public Material

### Shareable Dev Stories

1. **Problem -> Solution Journey**: User request -> implementation -> result
2. **Metrics Disclosure**: "How we improved D7 retention from X% to Y%"
3. **Failure Stories**: "Why this feature failed"
4. **Technical Deep Dives**: "How we built [feature] with SwiftUI"
5. **Indie Dev Journey**: Revenue, downloads, decision-making process
